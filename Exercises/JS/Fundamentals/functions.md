

# JavaScript Functions - Exercises

## Task 1: Function Declaration
Create a function using function declaration that returns the sum of two numbers.

## Task 2: Function Expression
Write a function expression that calculates the square of a number.

## Task 3: Arrow Function
Create an arrow function that checks if a number is even or odd.

## Task 4: Anonymous Function
Write an anonymous function assigned to a variable that converts Celsius to Fahrenheit.

## Task 5: Function Hoisting
Explain the difference between function declaration and function expression regarding hoisting. Create examples of both.

## Task 6: Default Parameters
Create a function declaration with default parameters that greets a user with a customizable message.

## Task 7: Rest Parameters
Write a function using rest parameters (`...args`) that calculates the average of multiple numbers.

## Task 8: Callback Function
Create a function that accepts a callback and uses it to process an array of numbers.

## Task 9: Higher-Order Function
Write a function that returns another function (closure) for multiplying numbers by a given factor.

## Task 10: IIFE (Immediately Invoked Function Expression)
Create an IIFE that logs a message and demonstrates variable scope encapsulation.

# JavaScript Loops - Exercises

## Task 1: Sum of Numbers
Write a program using a `for` loop to calculate the sum of numbers from 1 to 10.

## Task 2: Multiplication Table
Create a `while` loop that prints the multiplication table for number 5 (5×1 through 5×10).

## Task 3: Reverse String
Use a `for` loop to reverse a given string and display the result.

## Task 4: Count Vowels
Write a program using a `for...of` loop to count vowels in a sentence.

## Task 5: Factorial Calculation
Implement a `do...while` loop to calculate the factorial of a number.

## Task 6: Find Maximum
Use a `for` loop to find the maximum value in an array of numbers.

## Task 7: Filter Even Numbers
Create a loop to filter and display only even numbers from an array.

## Task 8: Nested Loop Pattern
Write nested `for` loops to create a 5×5 grid pattern using asterisks (*).

## Task 9: Break and Continue
Write a program demonstrating `break` and `continue` statements in a loop.

## Task 10: Array Transformation
Use a `for` loop to transform an array by doubling each element and storing results in a new array.
